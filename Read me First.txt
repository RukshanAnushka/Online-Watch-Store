		How To Run?

You’ll need XAMP Server or WAMP Server. 
Put the file inside “c:/xampp/htdocs/” or “c:/wamp/www/”.

“c:/xampp/htdocs/OnlineWatchStore” 
“c:/wamp/www/OnlineWatchStore”

Create a database and import sql file to run.
Go To “localhost/phpmyadmin” in any browser and create Database named “watch”.
In That Database, Import "watch.sql".

sql and finally open go to URL: “http://localhost/OnlineWatchStore/”
